RO.ME "3 DREAMS OF BLACK"

RO.ME is an interactive film, and honestly I have never seen anything like it before. This interactive music video/film allowes users to interact with the rendered envirenment in real time.
While song plays in the background, we journey in first person, user is able to see elements beyond what's in their realtime pov by hovering the mouse around.
Interaction is not limited to just shifting the POV either, there are elements that are rendered where mouse is hovered and in later parts of the video, we even see clever use of video effects shifting as user interacts with mouse. There is clever use of different velocities making the experience even more dynamic and interesting. (My favourite part was one where we follow a flock of birds.)
This project was developed in 2011, it is a pioneer project that showcases the endless possiblities of what can be created using WebGL and HTML.
